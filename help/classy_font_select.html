Font selection dialog

Description

The font dialog is used to select a font from a list of available fonts. The properties of the font such as point size, weight, slant, underline and overstrike can be selected using the entry and option boxes beside the listbox containing the fonts.

At the bottom of the dialog, a sample of the currently selected font is shown.

On the top, an entry showing the full font name is shown, this can also be edited directly. It is also useful to restore previously selected fonts using the defaults mechanism.

Action buttons

Action Select
The font you have selected now is ok.
Action Help
Pop up a help window
Action Close
Close the dialog box without selecting a file.
