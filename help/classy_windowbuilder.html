ClassyTcl Window Builder

The ClassyTcl window builder is used to change the layout of and code associated with dialogs, toplevels or frames. It is invoked by double clicking a window function in the Builder Window. When invoked, it pops up two windows: the work and the editing window

The work window

The work window displays an copy of the dialog being edited. The widgets on this copy do not exhibit their normal functions. They can be selected and A widget can be moved to another location by dragging.
