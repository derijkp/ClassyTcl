ColorSelect

subclass of Widget
A colorselector where the user can select a color using different methods: sample, HSV, RGB or colorname

ColorSelect specific options

Command-Line Name: -label
Database Name: ?
Database Class: ?
text on the label of the colorselector
Command-Line Name: -command
Database Name: command
Database Class: Command
command to be executed when the color is changed

ColorSelect specific methods

pathname set value
set current color to $value
pathname get
set current color
pathname select type
set the current type of selector
