OptionBox

subclass of Widget
creates an optionbox, a widget in which the user can select between different values using radiobuttons.

OptionBox specific options

Command-Line Name: -label
Database Name: label
Database Class: Label
Command-Line Name: -orient
Database Name: orient
Database Class: Orient
must be vertical or horizontal
Command-Line Name: -variable
Database Name: variable
Database Class: Variable

OptionBox specific methods

pathname add item text ?option value ...?
pathname set item
pathname get
pathname items
pathname button item
