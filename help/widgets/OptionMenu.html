OptionMenu

subclass of Widget
creates an optionmenu, a widget in which the user can select between different values using a menu.

OptionMenu specific options

Command-Line Name: -images
Database Name: images
Database Class: Images
This option can contain a list of images that will be displayed instead of text.
Command-Line Name: -command
Database Name: command
Database Class: Command
Command-Line Name: -list
Database Name: list
Database Class: List
