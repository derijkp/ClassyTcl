CanvasSeq

subclass of Widget
A CanvasSeq is not a widget type. It is a class whose objects can be associated with a canvas widget. When a CanvasSeq instance is associated with a canvas, it will draw on this canvas. Several CanvasSeq objects can be associated with the same canvas.

CanvasSeq specific options

BarcHart objects support the following options in its configure method
-canvas
name of canvas to draw the barchart
-area
area in which to diplay the chart (a list of four numbers)
-boxwidth
display a rectangle with this width showing the area
-boxcolor
color of area rectangle
