Themes configuration dialog

The themes dialog allows loading a number of related configuration settings (theme) from a file into the currently edited configuration in the Configuration dialog. It presents a list of themes available from the default places in the configuration system. A theme can also be loaded from a theme file at any place using the "Load theme from file" button.
  • Select theme: Load currently selected theme. You can also do this by double clicking on the theme
  • Load theme from file: Select a theme file (anywhere) to load
  • Delete theme: Delete the currently selected theme
