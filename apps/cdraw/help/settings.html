Structure settings

Menu:Options.Structure settings
Tcl routine:settings

Description

This dialog box determines the settings for the currently selected structure. Changing of these settings has no imediate effect. It is only noticable when parts of the structure are redrawn or when a new structure is drawn.

Options

Distance between

Bases (entry)
This is the general distance between to bases in loops.
Helix Bases (entry)
This is the distance between two adjacant bases in a helical segment.

Standard base pairs

Base distance (entry)
This is the distance between two complementary bases in a helical segment.
Line length (entry)
This is the length of the line between two complementary bases. If set to 0, a point will be drawn.
Line width (entry)
This is the width of the line between two complementary bases.

Non Standard base pairs

Show non standard base pairs (checkbutton)
If not checked, the line between non standard basepair will not be shown. The color, etc of non standard basepairs can also always be changed later.
Bulge distance (entry)
make non standard basepairs bulge out of the helix by the given amount.
Line length (entry)
This is the length of the line between two non standard bases. If set to 0, a point will be drawn.
Line width (entry)
This is the width of the line between non standard bases.

Auto connect

Treshhold (entry)
When the distance between two bases is larger than this treshhold, they will automatically be connected by a line.
Line width (entry)
This entry gives the width of the lines that will automatically connect bases that are far apart on the drawing.

Fonts

Base font (font select)
Gives the font used to display bases.
Helix font (font select)
Gives the font used to display Helix numbers.

Action buttons

Action Set
Change the settings of the current drawing.
Action Set as Default
Use the current settings as the default settings for new drawings
Action Defaults
Reset the settings to their default values.
Action Program defaults
Reset the settings to the program default values.
Action Help
What do you think?
Action Close
Close the dialog box without selecting a file.
