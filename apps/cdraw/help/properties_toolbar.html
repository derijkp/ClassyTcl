Properties Toolbar

Description

Each object has properties such as font, text, color, linewidth and position. The "Properties Toolbar" is a line of buttons that can be used to change these properties for all currently selected objects. This is the default secondary toolbar. The toolbar can be selected using the "Objects.Properties toolbar" menu, or the "Properties toolbar" button on the Main toolbar

Buttons

Adjust selection (transparant selection arrow)
The selection can be temporarily changed using the adjust selection dialog. This can be eg. used to only change the properties of a specific type of objects such as bases, base pairings, base numbers, helix names or labels.
Fill color
changes the main color of the objects.
Outline color
If an object has an outline (rectangle, oval), this button will change the color of this outline.
Font
changes the font of the objects.
Smooth
draw a smooth line
Cap style
style of the endpoints
Join style
Arrow
Arrow shape
Width
changes the line width for matching objects.
Text
changes the text for matching text objects.
Tags
change the tags.
