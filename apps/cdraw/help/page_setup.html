Page Setup

Description

The page setup dialog is used to change the current paper size, orientation and color.

Options

Paper color

Change the color of the paper

Paper size

This dialog defines the size of the paper to work on. This papersize does not have to be the same size as the final printout. For printing everything will be scaled to an appropriate size. In fact, for large structures it is often better to select a larger size. The standard papersizes can be selected from the optionmenu to the right. Non standard paper sizes can be entered in the 'Width' and 'Height' Optionbox: The paper can be oriented either with the long side vertical (Portrait) or horizontal (Landscape).
