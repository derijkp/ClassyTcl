Mode menu

Select
sets cdraw to the selection mode, where objects can be selected or edited
Rotate
Switch Rotation mode on or off.
Zoom
In zoom mode you can zoom in or out the drawing, and the Zoom toolbar is displayed.
Drawing objects
Several types of objects can be drawn starting from this menu:
  • Text: Create a Text object by clicking and typing in the text.
  • Line: Create a line by clicking on the page.
  • Polygon: Create a polygon by clicking on the page.
  • Rectangle: Create a rectangle by dragging on the page.
  • Oval: Create an oval object by dragging on the page.
  • Arc: Create an arc object by dragging on the page.
Insert bitmap
Insert a bitmap on the page
