Quickstart

Introduction

In short, RnaViz is a program that allows you to create 2D drawings of the structure embedded in a DCSE alignment file or a ct file. The layout of one drawing can be transfered to another drawing using skeleton files. Basic skeleton files are provided with the program for several molecules (5S, SSU, LSU, tRNA, Intron). The drawings can also be annotated easily.

The easiest help to get you started with RnaViz is the tutorial or howto section of the help files. It describes how to do most common tasks in RnaViz in a "cookbook" style.

You can find more general information about what RnaViz is and does in the general RnaViz help section and in the RnaViz article Text. Specific descriptions of all the options can be found in the contents.
