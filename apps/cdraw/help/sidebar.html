Sidebar

Description

The sidebar is a toolbar displayed to the left of the drawing page. From it, the different editing modes of RnaViz can be selected easily. The first part has buttons for the selection mode, RnaViz will behave differently when an object is clicked.
  • Select: select groups or objects
  • Select trees: select entire trees
  • Select structures: select entire helices
  • Select sub-structures: select helix segments
  • Select single objects: select individual objects
Rotate
Switch Rotation mode on or off.
Zoom
In zoom mode you can zoom in or out the drawing, and the Zoom toolbar is displayed.
Drawing objects
Several types of objects can be drawn starting from this menu:
  • Text: Create a Text object by clicking and typing in the text.
  • Line: Create a line by clicking on the page.
  • Polygon: Create a polygon by clicking on the page.
  • Rectangle: Create a rectangle by dragging on the page.
  • Oval: Create an oval object by dragging on the page.
  • Arc: Create an arc object by dragging on the page.
