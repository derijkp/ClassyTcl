The File menu

Open file
open files in one of the recognised formats.
Save
save current page to the current file as an RnaViz structure file.
Save as
Save the current page to a different file.
Print
lets you create a postscript file for printing.
Clear Page
Delete everything on the page.
Editor
invoke ClassyTcl text Editor
Command window
invoke the ClassyTcl command window
Builder
invoke the ClassyTcl builder to change this application

customise application
customise RnaViz.
customise menu
customise RnaViz menu.
customise toolbar
customise RnaViz toolbar.
Exit
Leave RnaViz
