Open file

Menu:File.Open Tcl routine:loadfile

Description

The file to be opened in RnaViz can be selected in a file selection box. Three types of files are recognised by RnaViz: ul>
  • ct format as used in mfold
  • DCSE alignment format
  • RnaViz structure file
    RnaViz will automatically detect which type of file must be opened, and take appropriate actions.
