Print dialog

Menu:File.Print Tcl routine:printcanvas

Description

This dialog is used to create a printable postscript file of the current page. The file created can be sent straigh to a postscript printer, or via ghostscript for non-postscript printers.

Options

File (Entry)
enter the name of the file to save the postscript to, or click on the browse button to use the file save dialog.
Print (Optionbox)
allows yo to select wether a color, grayscale or black and white print is produced.
Printsize
Here the size of the paper on which will be printed can set. If this size is different from the current paper size (for editing), everything will be scale appropriately. The standard papersizes can be selected from the optionmenu to the right. Non standard paper sizes can be entered in the 'Width' and 'Height' Action Save
Save the postscript file with the current settings
Action Help
What do you think?
Action Close
Close the dialog box without selecting a file.
